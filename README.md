# 🍔 FoodHub — Интернет-магазин еды

FoodHub — современный сайт доставки еды на React. Пользователи могут просматривать меню (данные из TheMealDB API), добавлять блюда в корзину и оформлять заказ.

## 🚀 Технологии

- **React 19** + Vite
- **React Router** — навигация между страницами
- **Axios** — запросы к API
- **React Icons** — иконки интерфейса
- **Framer Motion** — анимации карточек
- **Tailwind CSS** — стилизация и адаптивность

## 📄 Страницы

1. **Главная (`/`)** — баннер, преимущества, категории, популярные блюда
2. **Меню (`/menu`)** — каталог блюд из API с поиском и фильтром по категориям
3. **Корзина (`/cart`)** — список заказа, изменение количества, форма оформления
4. **О нас (`/about`)** — информация о компании, контакты, форма обратной связи
5. **404** — страница "не найдено"

## 🌐 API

TheMealDB API (https://www.themealdb.com/api.php) — бесплатное API рецептов и блюд (без ключа).

## 📦 Установка и запуск

```bash
npm install
npm run dev
```

Сборка для продакшена:

```bash
npm run build
```

## 📱 Адаптивность

Сайт полностью адаптивен для компьютеров, планшетов и телефонов (mobile-first на Tailwind CSS).

## 🎨 Дизайн

Тёплая палитра (оранжевый #FF6B35 + крем + тёмно-серый), карточки с тенями и hover-анимациями, шрифт Poppins.
